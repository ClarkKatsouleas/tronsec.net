Enterprise Network Security Homelab

A comprehensive enterprise-grade network security environment

Project Overview

Initial planning for this enterprise network security homelab began in March 2024, with physical implementation starting in August 2024. The network has grown from two initial devices to over seven, with plans for further expansion after graduating from Purdue University in May 2025. This project combines knowledge gained from independent study and undergraduate coursework at Purdue in Cybersecurity and Network Engineering Technology.

The lab incorporates enterprise-grade hardware including a Cisco Catalyst 3750-X switch, Cisco 1921 router, FortiNet FortiGate 40F Next Generation Firewall (NGFW), and an HP ProLiant DL360 Gen9 server with 256GB of RAM and 2TB of SSD storage. Additionally, two micro Dell OptiPlex servers (5060 and 5070) provide dedicated services with 16GB RAM each and combined 2TB storage. The network utilizes color-coded CAT6 cabling for different VLANs and implements comprehensive security practices including VLAN segmentation, RADIUS authentication, and Active Directory integration. Network Architecture & Segmentation

The network implements an enterprise-grade segmentation strategy using four primary VLANs, each with specific security policies and access controls applied. Open Shortest Path First is used as a dynamic routing protocol between the router and firewall within area 0. Network Address Translation is applied to all security zones off of the firewall for general internet connectivity. A breakdown of the VLANs as they relate to infrastructure within the network can be viewed below.

Management Infrastructure (VLAN 10)

  • Subnet: 10.26.10.0/24
  • Cable designation: Blue CAT6 UTP
  • Primary devices: Active Directory server (10.26.10.100), Management PC (10.26.10.26), HP ProLiant DL360 Gen9 ESXi Management Console (10.26.10.200)
  • RADIUS authentication integration for network devices
  • Primary DNS resolver for internal subnets and domain (tronsec.net)
  • Root Certificate Authority for the primary domain (10.26.10.100)
  • Link aggregation to HP ProLiant server using 4 x 1GbE CAT6 connections

Wireless Network (VLAN 20)

  • Subnet: 10.26.20.0/24
  • Cable designation: Red CAT6 UTP
  • Isolated DHCP services on Cisco 1921 router
  • External DNS resolution through Cisco OpenDNS (208.67.222.222)
  • No internal network access - internet gateway only
  • ACLs prevent all inter-VLAN communication
  • Managed through TP-Link AX1800 WiFi 6 Router

DMZ Services (VLAN 30)

  • Subnet: 10.26.30.0/24
  • Cable designation: Green CAT6 UTP
  • Hosts DMZ DNS server (10.26.30.100)
  • Relays internal DNS from Active Directory server
  • Direct internet access through FortiGate firewall
  • Selective access from VLAN 10 only
  • DNS forwarding to Cloudflare (1.1.1.1)
  • Services NTP for domain and log synchronization on networking devices

Server Infrastructure (VLAN 40)

  • Subnet: 10.26.40.0/24
  • Cable designation: Yellow CAT6 UTP
  • LibreNMS monitoring system (10.26.40.20)
  • SecurityOnion IDS/IPS platform (10.26.40.30)
  • Management Access restricted to VLAN 10 only

Physical Infrastructure

  • Cisco Catalyst 3750-X core switch with full VLAN support
  • Color-coded CAT6 cabling for easy VLAN identification
  • Link aggregation for server connectivity (4 x 1GbE)
  • FortiGate FG-40F NGFW for edge security and routing
  • Cisco 1921 router for internal routing and VLAN segmentation
  • TP-Link Wireless Router for mobile client connectivity

Network Connectivity & Routing

The network displays sophisticated routing and connectivity solutions including:

  • OSPF implementation between the Cisco 1921 router and FortiGate firewall for dynamic internal routing
  • Link Aggregation (LAG) for ESXi server connections, providing enhanced bandwidth and redundancy
  • Access Control Lists (ACLs) on both layer 2 and layer 3 devices for granular traffic control
  • RADIUS authentication integration for secure device access and user authentication

Active Directory & Authentication

A comprehensive authentication infrastructure built on Windows Server 2022:

  • Centralized Active Directory services for user and device management
  • RADIUS server integration for network device authentication
  • Certificate Authority services for internal PKI infrastructure
  • Group Policy implementation for security and access control
  • Integration with FortiGate NGFW for user-based access policies

Security Implementation

Multi-layered security approach incorporating:

  • FortiGate Next-Generation Firewall for advanced threat protection
  • Network segmentation with strict VLAN isolation
  • Layer 2 security including DHCP snooping and Dynamic ARP inspection
  • 802.1X port-based authentication for wired connections
  • Comprehensive logging and monitoring through SecurityOnion and LibreNMS

Virtualization Infrastructure

ESXi-based virtualization platform hosting critical services:

  • HP ProLiant DL360 Gen9 server with 256GB RAM and 4TB SSD storage
  • Network monitoring and security analysis virtual machines
  • Development and testing environments
  • Link aggregation for enhanced network performance
  • Integration with Active Directory for access control

Infrastructure Hardening

Comprehensive security hardening measures:

  • Secure device configuration baselines across all network equipment
  • Implementation of port security and unused port shutdown
  • Strong password policies and encryption standards
  • Regular security updates and patch management
  • Automated monitoring and alerting for security events
